# Aufgabe von Mailsniper

- Automatische Erkennung und Exploitation von Email Basierten Schwachstellen
- Spear Phishing und sammeln von Sensitiven Informationen Via Mail

# Features

- Automatische suche nach Emails, Email Ordnern und Sensitiven Daten in Microsoft Exchange Umgebungen
- Unterstüzt On-Prem und Cloud Based Infras
- Anpasbare Module für Gezieltes Phishing & Exfil

# Limitationen 

- Primär Fokus auf Microsoft Exchange & O365 Umgebungen
- Risiko von Email Security erkannt/geschnappt zu werden
- Ethische und Rechtliche Limitationen bei Unautorisierter Verwendung gegen ein Ziel

- **WICHTIG!!! Powershell Only!!!**
