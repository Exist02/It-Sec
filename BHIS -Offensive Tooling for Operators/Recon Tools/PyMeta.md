# Offensiver Einsatzzweck & Features
- Metadatenextraktion von Öffentlich Verfügbaren Dokumenten
- Identifizierung von Datenlecks mit Vertraulichen Informationen
- Automatisches suchen und Downloaden von Dokumenten eines Ziels
- Unterstüzt Unterschiedliche Dateitypen für Metadaten Analyse

#  Limitation
- Effizienz hängt von der Verfügbarkeit von Daten des Ziels ab
- Muss Genau analysiert werden um Unterschiede zwischen Intressanten und Belanglosen Daten zu Identifizieren
